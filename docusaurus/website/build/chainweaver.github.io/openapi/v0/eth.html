    GET /eth/{network}/addrs/{address}

    Parameters

    Name In Type Required Description
    network path string true none
    address path string true none
    before query integer(int32) false Filters response to only include transactions below before height in the blockchain.
    after query integer(int32) false Filters response to only include transactions above after height in the blockchain.
    limit query integer(int32) false limit sets the minimum number of returned TXRefs; there can be less if there are less than limit TXRefs associated with this address, but there can be more in the rare case of more TXRefs in the block at the bottom of your call. This ensures paging by block height never misses TXRefs. Defaults to 200, maximum is 2000.
    confirmations query integer(int32) false If set, only returns the balance and TXRefs that have at least this number of confirmations.

    network main

    network ropsten
